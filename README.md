# 🇺🇸 About

Hello! I'm Guilherme Paicheco, a Software Engineer from Brazil currently pursuing a Bachelor's Degree in Computer Science at IFMG while also studying Systems Analysis and Development at UNINTER.

I enjoy designing and building software that solves practical problems. My work ranges from artificial intelligence and automation to healthcare systems, productivity platforms and public sector solutions.

Rather than creating projects just to demonstrate technologies, I prefer building software with a clear purpose—applications that people can actually use, improve and rely on.
